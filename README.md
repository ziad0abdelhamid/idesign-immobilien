# DB ImmoDesign - Premium Real Estate Platform

A modern, fully-featured real estate website built with **Next.js 16**, **TypeScript**, **Tailwind CSS**, and **Supabase**. Features complete localization (English, Arabic, German), dark/light mode, glassmorphic design, and a powerful admin dashboard.

## ✨ Key Features

### 🌍 Full Localization

- **English (en)**, **Arabic (ar)** with RTL support, **German (de)**
- Automatic language switching with URL preservation
- All content translatable from admin dashboard

### 🎨 Design & UX

- **Glassmorphic navbar** that adapts on scroll
- **Premium color scheme** inspired by Pepsi branding
- **Dark/Light mode** with persistent storage
- Smooth animations via Framer Motion
- WCAG AA accessibility compliance

### 📱 Responsive & Mobile-First

- Fully responsive layout
- Touch-friendly interactive elements
- Optimized for all screen sizes

### 🏠 Property Management

- Beautiful property cards with image carousel
- Built-in payment calculator
- Real-time property information display
- No page navigation needed

### 👨‍💼 Admin Dashboard

- Full CRUD operations for properties
- Multi-language property content management
- Image upload and ordering
- Property status tracking (available/sold/rented)
- Clean, functional interface
